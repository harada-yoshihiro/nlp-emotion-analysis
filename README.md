# NLP恋愛感情分析プログラム

## 1. 概要
本プロジェクトでは、BERTモデルおよびWRIMEデータセットを用いて、  
文章中に含まれる恋愛感情を定量的に評価する自然言語処理プログラムを開発しました。

複数の感情要素を分解した上で加重平均を行い、  
「累積恋愛感情スコア」を定義・算出することで、  
告白文や恋愛表現の特徴を数値的に分析します。

---

## 2. 目的・成果
- 曖昧な恋愛感情を構造化し、定量分析が可能であるかを検証  
- Python / NLP / BERT を用いた実践的な分析プロセスを示す  
- Notebook・Pythonスクリプト・PDF形式の成果物をGitHub上に整理・公開  

---

## 3. 技術スタック
- **言語・環境**: Python 3 / Jupyter Notebook / Google Colab  
- **モデル**: bert-base-japanese-whole-word-masking  
- **ライブラリ**: pandas, scikit-learn, transformers, matplotlib  

---

## 4. ファイル構成
| ファイル名 | 内容 |
|------------|------|
| `nlp-emotion-analysis.ipynb` | 分析用Notebook |

---

## 5. 手法・工夫
1. 日本語文章の前処理（形態素解析・不要文字の除去）  
2. BERTによる文章単位の感情スコア推定  
3. 恋愛感情スコアの定義  
   - 喜び・期待・信頼など複数感情を加重平均  
4. 文例ごとのスコア比較および可視化  

---

## 6. 実験結果
- 告白文や恋愛表現を含む文章では、恋愛感情スコアが高く算出される傾向を確認できた
- 情景描写文においては誤判定が見られ、感情文脈理解の課題が示唆された

---

## 7. 学び・考察
- 曖昧な感情表現であっても、分解・定義を行うことで技術的に扱える可能性を確認できた
- 感情スコアの重み付けは手動で定義しており、機械学習による最適化の余地が残る
- NLPを用いた課題設定から分析、改善検討までの一連のプロセスを実践できた

---

## 8. 注意事項
- 本プログラムの実行には WRIME データセットが必要です
